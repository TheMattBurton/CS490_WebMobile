Repository Report

Before you can create a repository, you will need:

  • Internet browser
  • GitHub account
  • GitHub Desktop application

These are the steps in creating the repository:

  1. Click the '+' dropdown icon in the upper-right corner of the screen shown below.
  2. Select "New Repository"
  3. Give the repository a name and click Complete.
  4. Click Clone or download as shown below
  5. Follow prompts in GitHub Desktop
